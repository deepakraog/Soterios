ABOUT THIS APPLICATION

Soterios System Tools is a local-first utility for system maintenance,
monitoring, and basic security checks (file scanning, password tools,
disk/CPU/process monitoring, and opt-in maintenance scripts).

It does not collect, transmit, or upload any data. All scanning and
analysis happens entirely on your own machine. No telemetry, no analytics,
no network calls of any kind.

The file scanner is a local heuristic/signature tool, not a substitute for
a dedicated antivirus product.
